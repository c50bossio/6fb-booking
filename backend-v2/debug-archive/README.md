# Debug Archive

This directory contains debug and test files created during development and troubleshooting.
These files are kept for reference but are not part of the production codebase.

## File Categories

### Registration/Email Debugging
- Various test scripts for registration flow
- SendGrid email configuration tests
- Puppeteer browser automation tests

### Analytics Debugging
- API test scripts for analytics endpoints
- Authentication verification scripts

### Test HTML Files
- Direct form submission tests
- Registration form tests

## Note
These files should be reviewed periodically and removed when no longer needed.
They are intentionally not tracked in git to keep the repository clean.
